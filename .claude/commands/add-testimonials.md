# Add Testimonials Section

Add a testimonials / social proof section to the current client project.

Ask the user:
1. How many testimonials to show? (3-6 recommended)
2. For each: name, location (city/country), quote text, optional star rating (1-5)
3. Section title text

Then create a `Testimonials.jsx` section with:
- Cards using glassmorphism: `bg-white/5 backdrop-blur-sm border border-white/10 rounded-2xl`
- Quote marks styled in amber/gold accent color
- Star rating displayed as ★ symbols if provided
- Responsive grid: 1 col mobile, 2-3 cols desktop
- Dark background consistent with the project theme
- Import and add to main page in the correct position
